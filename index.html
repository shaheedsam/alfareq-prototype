IIS Hosting Setup Guide

1

Locate the Project Folder

Ensure all your prototype files (like index.html, employee_registration_form.html, and unified_employee_registration_prototype.html) are kept in a single directory on your machine. Currently they are located at:

D:\Activities\Experiments
2

Open IIS Manager

Press Win + R, type inetmgr, and press Enter to launch the Internet Information Services (IIS) Manager.

3

Create a New Website or Application

In the left Connections panel, expand your server node, right-click on Sites, and select Add Website....

Fill in the settings:

  • Site name: EmployeePrototypes
  • Physical path: D:\Activities\Experiments
  • Binding Port: Choose an open port (e.g. 8082 or 8888) to avoid conflicts.
4

Verify and Present

Click OK to start the site. Open your web browser and navigate to:

http://localhost:8082

Your client can now see the presentation hub directly on the local network using your machine's local IP address (e.g. http://192.168.x.x:8082).
